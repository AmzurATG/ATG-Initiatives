# Lesson Plan Assignment: Deep Learning Foundations

---

## Assignment Information

**Category:** Category 1: Foundational AI & ML Concepts  
**Subcategory:** 1.2 Deep Learning Foundations  
**Assigned To:** [Team Member 1 & Team Member 2]  
**Presentation Date:** [Date]  
**Duration:** 30-45 minutes

---

## Learning Objectives

By the end of this session, team members should be able to:
1. Explain what neural networks are and how they differ from traditional ML
2. Understand basic neural network architectures (CNNs, RNNs, Transformers)
3. Recognize common training challenges and solutions (vanishing gradients, learning rate, etc.)

---

## Topics to Cover

- **Neural Networks Basics:** Neurons, layers, activation functions, forward/backward propagation
- **Common Architectures:** CNNs for images, RNNs for sequences, Transformers for NLP
- **Training Dynamics:** Loss functions, optimizers (SGD, Adam), learning rate scheduling
- **Transfer Learning:** Pre-trained models, fine-tuning, feature extraction

---

## Assignment Deliverables

### 1. Presentation (Required)
- Create slides explaining neural network concepts with visual diagrams
- Include architecture diagrams for CNN, RNN, and Transformer
- Duration: 20-30 minutes

### 2. Code Demo/Example (Required)
- Build a simple neural network using PyTorch or TensorFlow
- Demonstrate transfer learning with a pre-trained model (e.g., ResNet, BERT)
- Show before/after comparison of model performance

### 3. Resources & References (Required)
- Curate 3-5 quality resources (3Blue1Brown videos, PyTorch tutorials, papers)
- Create a quick reference for common architectures and use cases
- Share links to pre-trained model repositories (Hugging Face, TensorFlow Hub)

### 4. Q&A Session (Required)
- Prepare for 10-15 minutes of questions
- Discussion questions:
  - "When should we use deep learning vs traditional ML?"
  - "Which architecture fits our specific use case?"

## Preparation Guidelines

- **Research:** Explore PyTorch/TensorFlow documentation and tutorials
- **Practice:** Train a simple model and experiment with hyperparameters
- **Collaborate:** Split architecture types between team members
- **Engage:** Use visual animations to show how neural networks learn
