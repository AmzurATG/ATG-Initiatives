# Lesson Plan Assignment: Enterprise Use Cases

---

## Assignment Information

**Category:** Category 11: Domain-Specific Applications  
**Subcategory:** 11.1 Enterprise Use Cases  
**Assigned To:** [Team Member 1 & Team Member 2]  
**Presentation Date:** [Date]  
**Duration:** 30-45 minutes

---

## Learning Objectives

By the end of this session, team members should be able to:
1. Identify AI opportunities in enterprise scenarios
2. Design solutions for customer support automation
3. Understand document intelligence and code generation applications

---

## Topics to Cover

- **Customer Support Automation:** Chatbots, ticket routing, sentiment analysis
- **Document Intelligence:** Extraction, classification, summarization at scale
- **Code Generation & Review:** AI-assisted development, code review automation
- **Business Process Automation:** Workflow automation, decision support

---

## Assignment Deliverables

### 1. Presentation (Required)
- Create slides showcasing enterprise AI use cases
- Include ROI examples and implementation strategies
- Duration: 20-30 minutes

### 2. Code Demo/Example (Required)
- Build enterprise use case demo (e.g., support chatbot)
- Demonstrate document processing pipeline
- Show code review automation example

### 3. Resources & References (Required)
- Curate 3-5 case studies of successful implementations
- Create use case templates for enterprise scenarios
- Share ROI calculation frameworks

### 4. Q&A Session (Required)
- Prepare for 10-15 minutes of questions
- Discussion questions:
  - "Which use cases apply to our clients?"
  - "How do we pitch these solutions effectively?"
