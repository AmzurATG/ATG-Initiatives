# Lesson Plan Assignment: Internal Tools & Productivity

---

## Assignment Information

**Category:** Category 11: Domain-Specific Applications  
**Subcategory:** 11.3 Internal Tools & Productivity  
**Assigned To:** [Team Member 1 & Team Member 2]  
**Presentation Date:** [Date]  
**Duration:** 30-45 minutes

---

## Learning Objectives

By the end of this session, team members should be able to:
1. Build AI-powered internal productivity tools
2. Implement knowledge management systems with AI
3. Create AI assistants for team workflows

---

## Topics to Cover

- **Knowledge Management:** Internal wikis, documentation search, Q&A systems
- **Developer Tools & Assistants:** Code helpers, debugging assistants, DevOps automation
- **Meeting Summarization:** Transcription, action items, note-taking automation
- **Search & Discovery:** Semantic search across company resources

---

## Assignment Deliverables

### 1. Presentation (Required)
- Create slides on internal AI tool opportunities
- Include productivity impact examples
- Duration: 20-30 minutes

### 2. Code Demo/Example (Required)
- Build internal knowledge management system
- Demonstrate meeting summarization tool
- Show developer productivity assistant

### 3. Resources & References (Required)
- Curate 3-5 resources on internal AI tools
- Create roadmap for ATG internal AI tools
- Share implementation templates

### 4. Q&A Session (Required)
- Prepare for 10-15 minutes of questions
- Discussion questions:
  - "What internal tools would help ATG most?"
  - "How do we measure productivity improvement?"

- **Research:** Study internal AI tool implementations
- **Practice:** Build tools addressing real ATG needs
- **Collaborate:** Each build different tool type
- **Engage:** Survey team for productivity pain points
