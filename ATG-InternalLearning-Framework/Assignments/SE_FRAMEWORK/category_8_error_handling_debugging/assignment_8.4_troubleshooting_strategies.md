4. Debug issues in production environments

---

## Topics to Cover

### 1. Systematic Debugging Approach
- Scientific method for debugging
- Hypothesis-driven investigation
- Divide and conquer strategy
- Reproduction steps importance
- Minimal reproducible examples
- Documentation of findings

### 2. Log Analysis Techniques
- Log aggregation and search
- Pattern recognition in logs
- Correlation ID tracing
- Log level filtering
- Structured log querying
- Tools: ELK, Splunk, CloudWatch

### 3. Root Cause Analysis
- 5 Whys technique
- Fishbone (Ishikawa) diagrams
- Distinguishing symptoms from causes
- Contributing factors vs root cause
- Preventing recurrence
- Post-mortem documentation

### 4. Debugging in Production
- Safety considerations
- Read-only investigation
- Feature flags for isolation
- A/B testing for comparison
- Canary analysis
- War room protocols
- Communication during incidents
