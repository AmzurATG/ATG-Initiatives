## Learning Objectives
By the end of this session, the team should be able to:
1. Understand GitHub Copilot capabilities and limitations
2. Write effective prompts for better suggestions
3. Use Copilot Chat effectively
4. Leverage context for accurate completions

### 2. Writing Effective Prompts
- Natural language comments
- Function signature design
- Context setting
- Example-driven prompts
- Breaking down complex tasks
- Iterative refinement

### 3. Copilot Chat
- Asking questions about code
- Requesting explanations
- Getting implementation guidance
- Slash commands (/explain, /fix, /tests)
- Context selection
- Multi-turn conversations

### 4. Leveraging Context
- Open files provide context
- Code in the same file
- Imported modules
- Recently edited files
- Documentation comments
- Test files for implementation
