- Sequence diagrams
- Entity-relationship diagrams
- Flow charts

- Consistent notation

- Legend and key
- Visual hierarchy

### 3. Diagramming Tools
- Draw.io/Diagrams.net
- Lucidchart
- Miro for collaboration
- Excalidraw for sketching
- Specialized tools (dbdiagram.io)
- Tool selection criteria

### 4. Diagram as Code
- Mermaid syntax
- PlantUML
- Structurizr for C4
- Version control benefits
- Automated generation
- Integration with docs
